### Step 2: 커스텀 동적 장애물 환경 구현
`robosuite`의 `TableTop` 환경을 상속받아, `step` 함수 호출 시마다 장애물의 위치를 업데이트하는 코드를 작성합니다.



**핵심 코드 스켈레톤:**
```python
import numpy as np
from robosuite.environments.manipulation.table_top import TableTop
from robosuite.models.objects import SphereObject
from robosuite.utils.mjcf_utils import CustomMaterial

class DynamicObstacleEnv(TableTop):
    def __init__(self, **kwargs):
        # 1. 장애물 객체 정의 (빨간색 구체)
        self.obstacle = SphereObject(
            name="dynamic_obs",
            size=[0.05],
            rgba=[1, 0, 0, 1],
            friction=[0.0001, 0.005, 0.0001]
        )
        super().__init__(**kwargs)

    def _load_model(self):
        super()._load_model()
        # 2. 테이블 위에 장애물 배치
        self.model.place_visual_obj(self.obstacle, target_pos=[0, 0, 1.0])

    def step(self, action):
        # 3. 시간에 따른 장애물 위치 업데이트 (예: x축 왕복 운동)
        new_x = 0.4 * np.sin(self.cur_time * 2.0)
        self.sim.model.body_pos[self.sim.model.body_name2id("dynamic_obs")] = [new_x, 0, 1.0]
        
        return super().step(action)
```

### Step 3: Observation 데이터 확인
학습에 필요한 상태 값이 정상적으로 출력되는지 확인합니다.
* `obs['robot0_eef_pos']`: 로봇 말단 위치
* `obs['dynamic_obs_pos']`: 장애물 위치
* `obs['dynamic_obs_vel']`: 장애물 속도 (이전 프레임과의 차이로 계산 권장)
