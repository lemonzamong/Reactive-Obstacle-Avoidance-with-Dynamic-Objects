# 📅 [Week 3] 모방 학습(BC) 기반 베이스라인 모델 구축

**목표:** `robomimic`을 활용하여 전문가 데이터셋을 학습시키고, 장애물을 회피하여 물체를 옮기는 기초 정책(Policy)을 생성한다.

---

## 1. 학습 알고리즘 설정 (Algorithm Setup)
가장 안정적이고 범용적인 **BC-RNN (Behavioral Cloning with RNN)**을 사용합니다. RNN은 시계열 정보를 처리하므로, 장애물의 움직임을 예측하며 회피하는 데 유리합니다.

* **Framework:** `robomimic`
* **Algorithm:** BC-RNN (LSTMs / GRUs 기반)
* **Input:** Low-dim observations (말단 위치, 장애물 위치/속도 등)
* **Output:** Joint Velocities 또는 OSC(Operational Space Control) Action

---

## 2. 주요 작업 단계 (Step-by-Step)

### Step 1: `robomimic` 데이터 변환 및 설정
`robosuite`에서 수집한 `.hdf5` 파일을 `robomimic`이 학습할 수 있는 표준 포맷으로 변환하고 설정 파일(`config`)을 준비합니다.
* **Action:** `python robomimic/scripts/dataset_states_to_obs.py`를 사용하여 raw states를 observations로 변환.
* **Config:** `robomimic/exps/templates/bc_rnn.json` 파일을 복사하여 우리 프로젝트의 키값(`dynamic_obs_pos` 등)에 맞게 수정.

### Step 2: 모델 학습 실행 (Training)
실제 GPU(또는 CPU)를 사용하여 신경망을 최적화합니다.

* **Command:** `python robomimic/scripts/train.py --config path/to/bc_rnn.json`
* **Monitoring:** 학습 도중 Loss(손실 함수)가 떨어지는지, 그리고 중간중간 시뮬레이션에서 'Rollout(테스트)' 성공률이 올라가는지 확인합니다.

### Step 3: 모델 평가 및 비판적 분석 (Evaluation)
학습된 체크포인트(`.pth`) 파일을 불러와 실제 환경에서 구동해 봅니다.
* **Action:** 다양한 장애물 속도 조건에서 로봇이 충돌 없이 미션을 완수하는지 테스트.
* **Check:** 로봇이 장애물 근처에서 '머뭇거리거나' 혹은 '과감하게 피하는지' 관찰하고, 사람의 시연과 비교하여 어떤 부분이 부족한지 파악합니다.
